[[https://en.wikipedia.org/wiki/CHIP-8][CHIP-8]] emulator using [[https://www.raylib.com/][raylib]]. It expects =*.ch8= file.

#+begin_src shell
  cargo run -- -r foo.ch8
#+end_src

Tested with [[https://github.com/Timendus/chip8-test-suite][chip8-test-suite]] and few games.

[[./outlaw.png]]

Screenshot above is from the [[https://johnearnest.github.io/chip8Archive/play.html?p=outlaw][Outlaw]] game running on this emulator.

CHIP-8 keypad -> keyboard mapping

| 1 | 2 | 3 | C | -> | 1 | 2 | 3 | 4 |
| 4 | 5 | 6 | D | -> | Q | W | E | R |
| 7 | 8 | 9 | E | -> | A | S | D | F |
| A | 0 | B | F | -> | Z | X | C | V |
